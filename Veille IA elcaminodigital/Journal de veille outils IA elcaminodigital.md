# Journal de veille outils

## 2026-05-19

### Storyflow
- Lien : https://storyflow.so
- Description : Canvas visuel IA qui lit toute ta strategie (brand, persona, plan marketing) en une fois. Le seul outil ou l'IA lit tout le canvas + 3 documents + 1 tactic dans le meme contexte.
- Utilite soloentrepreneur : Remplace 3-4 outils (tableau blanc, doc marque, board projet, chat IA) avec un seul. Ideal pour garder brand + persona + plan marketing ensemble.
- A recommander (1-5) : 5

### ChatGPT / Claude
- Lien : https://chat.openai.com / https://claude.ai
- Description : Assistants IA polyvalents pour redaction, strategie, recherche, ideaison. Ton equipe marketing dans une fenetre de chat.
- Utilite soloentrepreneur : Blog posts, emails, ads, reseaux sociaux, pages de vente - tout depuis un outil. ChatGPT meilleur pour le volume, Claude meilleur pour les docs longs (200K+ tokens).
- A recommander (1-5) : 5

### Canva
- Lien : https://canva.com
- Description : Outil de design browser-based avec IA pour generer des images, supprimer des backgrounds, creer des designs depuis des prompts.
- Utilite soloentrepreneur : Assets marketing, graphiques reseaux, presentations, mockups produits - visuels pros sans competences design.
- A recommander (1-5) : 5

### Calendly
- Lien : https://calendly.com
- Description : Reservation self-service. Les clients bookent directement via un lien partageable avec tes disponibilites.
- Utilite soloentrepreneur : Pour demos, appels ventes, sessions support - les clients bookent eux-memes au lieu de t'envoyer des mails.
- A recommander (1-5) : 5

### Zapier
- Lien : https://zapier.com
- Description : Plateforme d'automatisation qui connecte tes apps et cree des workflows sans code.
- Utilite soloentrepreneur : La glue qui tient ton business ensemble. Nouveau client ? Ajout auto au CRM, email de bienvenue, notification, maj spreadsheet - sans lever le petit doigt.
- A recommander (1-5) : 4

### Notion
- Lien : https://notion.so
- Description : Workspace pour docs, wikis, databases, enrichi par IA pour rediger, resumer et repondre a des questions.
- Utilite soloentrepreneur : Le systeme d'exploitation de ton business. SOPs, suivi projet, base clients, calendar contenu - tout dans un endroit searchable et boosté IA.
- A recommander (1-5) : 4

### Carly
- Lien : https://carlyassistant.com
- Description : Assistant IA de scheduling avec 70+ integrations. Gere le scheduling par email et texto - tu forwardes la demande, CC Carly, et il coordonne comme un assistant humain.
- Utilite soloentrepreneur : Elimine les allers-retours de RDV sans embaucher une VA. Pages de booking gratuites aussi.
- A recommander (1-5) : 5

### Opus Clip
- Lien : https://opusclip.com
- Description : Transforme des videos longues en clips courts optimises pour les reseaux sociaux avec captions automatiques.
- Utilite soloentrepreneur : Enregistres un podcast ou video YouTube et obtiens une semaine de clips verticaux prets pour TikTok, Reels, Shorts.
- A recommander (1-5) : 5

### Cursor / Replit
- Lien : https://cursor.sh / https://replit.com
- Description : Environnements de coding IA qui permettent de construire du software depuis des descriptions en language naturel.
- Utilite soloentrepreneur : Pas besoin de cofounder technique. Les non-developpeurs peuvent creer des produits fonctionnels.
- A recommander (1-5) : 4

### Attio
- Lien : https://attio.com
- Description : CRM qui sync automatiquement email et calendrier, enrichit les contacts, et surface des insights relationnels sans saisie manuelle.
- Utilite soloentrepreneur : Les CRMs traditionnels demande de la discipline. Attio se maintient tout seul - il tire de l'email/calendrier, enrichit auto, et te montre qui a besoin d'attention.
- A recommander (1-5) : 4

### Crisp
- Lien : https://crisp.chat
- Description : Plateforme de messaging client avec chatbot IA a un prix plus bas qu'Intercom.
- Utilite soloentrepreneur : Reponses IA, inbox partage, base de connaissances - pricing adapte aux solopreneurs.
- A recommander (1-5) : 4

### Intercom + Fin
- Lien : https://intercom.com
- Description : Plateforme de messaging client avec agent IA (Fin) qui resout les conversations de support de maniere autonome.
- Utilite soloentrepreneur : Support 24/7 sans embaucher. Fin apprend de tes docs d'aide et conversations passees.
- A recommander (1-5) : 4

### Clay
- Lien : https://clay.com
- Description : Plateforme d'enrichissement de donnees et outreach qui tire de dizaines de sources pour construire des profils de leads et personaliser l'outreach a l'echelle.
- Utilite soloentrepreneur : Automatise la recherche de prospects qui prendrait des heures par lead.
- A recommander (1-5) : 3

### Lindy
- Lien : https://lindy.ai
- Description : Permet de creer des agents IA personnalises qui automatisent les workflows avec prise de decision - sans code.
- Utilite soloentrepreneur : Au-dela de l'automatisation simple, les agents Lindy peuvent prendre des decisions - triage emails, qualification leads, moderation contenu.
- A recommander (1-5) : 3

### Gamma
- Lien : https://gamma.app
- Description : Cree des presentations, pitch decks et documents depuis des prompts.
- Utilite soloentrepreneur : Decks investisseurs, presentations ventes, demos produits qui ont l'air professionnels - sans competences design.
- A recommander (1-5) : 4

### Jasper
- Lien : https://jasper.ai
- Description : Outil d'ecriture IA built specifically pour le marketing, avec templates pour ads, emails, landing pages, contenu social + entrainement brand voice.
- Utilite soloentrepreneur : Quand tu as besoin de volume - multiples variations d'ads, posts blog hebdo, contenu social quotidien - les templates et brand voice speed up la production.
- A recommander (1-5) : 3

### Perplexity
- Lien : https://perplexity.ai
- Description : Outil de recherche IA qui repond aux questions avec infos sourcees - plus rapide que rechercher, plus fiable que l'IA generale pour les queries factuelles.
- Utilite soloentrepreneur : Recherche concurrents, sizing marche, tendances industrie, questions techniques - responses sourcees en secondes.
- A recommander (1-5) : 4

### Kick
- Lien : https://kick.co
- Description : Outil comptable IA-native qui categorise auto les transactions et gere la compta avec minimum d'input.
- Utilite soloentrepreneur : La compta se fait en background, pas comme tache hebdomadaire. L'IA apprend tes categories et ne te surface que ce qui necessite ton attention.
- A recommander (1-5) : 4

### Grain / Gong
- Lien : https://grain.com / https://gong.io
- Description : Outils d'enregistrement et analyse d'appels qui transcript, resument et extraient des insights des appels clients.
- Utilite soloentrepreneur : Chaque appel client contient de l'or - objections, demandes de features, mentions concurrents, signaux d'achat. Ces outils capturent tout et rendent searchable.
- A recommander (1-5) : 3

### Figma + Relume
- Lien : https://figma.com / https://relume.io
- Description : Figma est l'outil de design digital standard. Relume utilise l'IA pour generer wireframes et copy de site qui exportent vers Figma ou Webflow.
- Utilite soloentrepreneur : De l'idee aux pages de site designed en heures, pas en semaines - Relume genere structure et copy, Figma affine.
- A recommander (1-5) : 4

### Make (Integromat)
- Lien : https://make.com
- Description : Alternative a Zapier pour l'automatisation avec plus de flexibilite et pricing souvent plus avantageux.
- Utilite soloentrepreneur : Connecte tes apps et cree des workflows complexes sans code.
- A recommander (1-5) : 4

### Heyron
- Lien : https://heyron.ai
- Description : Assistant IA personnel qui s'integre a ton workflow quotidien (mails, messages, taches). Tout comme un assistant humain, il apprend de tes habitudes et automatise les actions repetitives.
- Utilite soloentrepreneur : Gain de temps enorme sur la gestion des mails et messages. Tout se fait via chat, sans avoir a configurer des integrations complexes.
- A recommander (1-5) : 5
